## âš¡ Getting Started

### Prerequisites
- Node.js (v16+ recommended)
- npm or yarn

### Installation
```bash
git clone https://github.com/fadigatardzenyuy/ShisongConnect.git
cd ShisongConnect
npm install
```

### Running the App
```bash
npm run dev
```
Visit `http://localhost:5173` in your browser.

### Building for Production
```bash
npm run build
```
